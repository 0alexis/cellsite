┌──────────────────────────────────────────────────────────────────────────────┐
│ TROUBLESHOOTING RAPIDO                                                       │
└──────────────────────────────────────────────────────────────────────────────┘

    Problema                   Solucion
    ────────────────────────   ──────────────────────────────
    Python no found        → Instalar Python con "Add to PATH"
    No templates found     → Verificar carpeta templates/ existe
    Build fails            → Ejecutar verify_build_ready.bat
    Start_app no abre      → Run en CMD para ver errores
    Permission denied      → Permisos carpeta o USB readonly
    Exe tarda en iniciar   → Normal en USB 2.0, usa USB 3.0
    Falta icono            → Crear ICON_CELLSITE.ico (.256x256px)


┌──────────────────────────────────────────────────────────────────────────────┐
│ ARCHIVOS PRINCIPALES CREADOS                                                 │
└──────────────────────────────────────────────────────────────────────────────┘

    build_portable.bat
    └─ Script de construccion principal
       └─ Crea entorno virtual, instala deps, compila con PyInstaller

    start_app.bat
    └─ Script para iniciar app compilada
       └─ Verifica CellSite.exe, crea carpeta data/, abre navegador

    verify_build_ready.bat
    └─ Script de diagnostico pre-construccion
       └─ Verifica Python, archivos, dependencias, espacio disco

    menu.bat
    └─ Menu interactivo principal
       └─ Opciones: iniciar, construir, verificar, abrir docs

    CellSite_portable.spec
    └─ Configuracion PyInstaller
       └─ Define como empaquetar app (onefile, add-data, etc.)

    Documentacion (TXT):
    ├─ README_CONSTRUCCION_USB.txt        (resumen para empezar)
    ├─ INSTRUCCIONES_USB_PORTABLE.txt     (para usuarios finales)
    ├─ WORKFLOW_CONSTRUCCION_USB.txt      (procedimiento detallado)
    └─ README.md                          (actualizado)
