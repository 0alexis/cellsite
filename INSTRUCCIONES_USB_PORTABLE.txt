📦 CONTENIDO DE ESTA CARPETA
═════════════════════════════════════════════════════════════════════════════════

La carpeta CellSite_portable contiene:

  📁 CellSite.exe               - Aplicacion ejecutable (autocontenida)
  📄 start_app.bat              - Script para iniciar la aplicacion
  📄 install_autostart.bat      - Activa inicio automatico con Windows
  📄 uninstall_autostart.bat    - Quita inicio automatico con Windows
  📁 data/                      - Carpeta con base de datos (app.db)
  📁 templates/                 - Plantillas HTML
  📁 static/                    - Archivos CSS e imagenes


🚀 COMO USAR DESDE USB
═════════════════════════════════════════════════════════════════════════════════

1. COPIAR A USB
   ➜ Copia toda la carpeta CellSite_portable a tu USB
   
2. INICIAR LA APLICACION
   ➜ Abre el USB en Windows
   ➜ Haz doble-clic en: start_app.bat
  ➜ Al abrirse una vez, CellSite queda registrado para iniciar solo con Windows
   
3. ESPERAR A QUE INICIE
   ➜ La primera vez tarda 5-10 segundos
   ➜ Se abrira automáticamente en tu navegador
  ➜ Si no se abre, entra a: http://127.0.0.1:8765

4. LOGIN
   ➜ Usuario: admin
   ➜ Contraseña: admin
   ➜ Puedes crear mas usuarios desde la app

5. INICIO AUTOMATICO CON WINDOWS
  ➜ CellSite se registra solo la primera vez que se ejecuta el .exe portable
  ➜ Desde ese momento se abrira solo al iniciar sesion en Windows
  ➜ Si cambias la carpeta o cambia la letra del USB, ejecuta install_autostart.bat otra vez
  ➜ Para quitarlo del inicio automatico, ejecuta: uninstall_autostart.bat

📊 TUS DATOS - UBICACIÓN Y RESPALDO
═════════════════════════════════════════════════════════════════════════════════

DONDE ESTAN TUS DATOS:
  → USB:/CellSite_portable/data/app.db

COMO HACER BACKUP:
  1. Abre la carpeta: USB:/CellSite_portable/data/
  2. Copia el archivo: app.db
  3. Guardalo en: tu computadora, nube, otro USB, etc.

COMO RESTAURAR DESDE BACKUP:
  1. Apaga CellSite
  2. Reemplaza: USB:/CellSite_portable/data/app.db
  3. Con tu archivo de respaldo
  4. Reinicia CellSite


⚠️ IMPORTANTE - PERMISOS DE LECTURA/ESCRITURA
═════════════════════════════════════════════════════════════════════════════════

Si obtienes error: "Permission denied" o "Cannot write to database"

SOLUCION EN WINDOWS:
  1. Clic derecho en la carpeta CellSite_portable
  2. Propiedades
  3. Pestaña "Seguridad"
  4. Clic en tu usuario
  5. Clic en "Editar..."
  6. Marca "Control total"
  7. Aplicar y Aceptar

O si es USB:
  1. Conecta el USB a la computadora
  2. Clic derecho en el USB
  3. Propiedades
  4. Desmarca "Solo lectura" (si esta marcado)
  5. Aplicar

🐛 SOLUCIÓN DE PROBLEMAS
═════════════════════════════════════════════════════════════════════════════════

PROBLEMA: "Acceso denegado al ejecutar start_app.bat"
SOLUCIONES:
  1. Haz clic derecho en start_app.bat → "Run as administrator"
  2. Si aun falla, edita start_app.bat con Bloc de notas
  3. Cambia "%SCRIPT_DIR%CellSite.exe" por la ruta completa del USB

PROBLEMA: El navegador no se abre automáticamente
SOLUCION:
  1. Abre manualmente: http://127.0.0.1:8765
  2. Si no carga, espera 10 segundos mas
  3. Asegurate que la ventana del cmd siga abierta (app en ejecucion)

PROBLEMA: "CellSite.exe no encontrado"
SOLUCION:
  1. Verifica que estes en la carpeta correcta (CellSite_portable)
  2. Haz doble-clic en start_app.bat, NO en CellSite.exe directamente
  3. Recopia todos los archivos del USB

PROBLEMA: Base de datos corrupta o no carga
SOLUCION:
  1. Busca: data/app.db.backup_*
  2. Renombralo a: app.db
  3. Reinicia CellSite


💡 CONSEJOS Y TRUCOS
═════════════════════════════════════════════════════════════════════════════════

1. CREAR ACCESO RAPIDO EN ESCRITORIO:
   - Clic derecho en start_app.bat
   - "Enviar a" → "Escritorio (crear acceso directo)"
   - Ahora puedes iniciar desde el escritorio

2. USAR MULTIPLES USB:
   - Copia la carpeta CellSite_portable a cada USB
   - Todos tendran la misma app y datos iniciales
   - Para sincronizar: copia manualmente data/app.db entre USBs

3. ACTUALIZAR LA APLICACION:
   - Si hay nuevas versiones, construye un nuevo CellSite.exe
   - Reemplaza solo CellSite.exe, templates/ y static/
   - Deja la carpeta data/ intacta (tus datos)

4. VELOCIDAD:
   - En USB 3.0 es significativamente mas rapido que USB 2.0
   - SSD USB es la opcion ideal


📱 USO DESDE TELEFONO/TABLET EN LA MISMA RED
═════════════════════════════════════════════════════════════════════════════════

Si quieres usar CellSite desde otro dispositivo en la red:

1. Identifica la IP de tu computadora:
   - Abre cmd.exe
   - Escribe: ipconfig
   - Busca "IPv4 Address" (ej: 192.168.1.100)

2. En tu telefono/tablet ingresa:
  - http://[IP_DE_TU_PC]:8765
  - Ejemplo: http://192.168.1.100:8765

NOTA: Solo funciona si estan en la MISMA red wifi
