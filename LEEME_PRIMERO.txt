              CellSite - Sistema de Ventas
═══════════════════════════════════════════════════════════════

📱 Sistema completo para gestión de ventas de celulares
   con pagos en cuotas y control de deudores.


═══════════════════════════════════════════════════════════════
  🚀 INICIO RÁPIDO
═══════════════════════════════════════════════════════════════

🪟 WINDOWS:
   1. Abre CMD (Símbolo del sistema)
   2. Navega a esta carpeta: cd C:\ruta\a\CellSite
   3. Ejecuta: build.bat
   4. Espera 3-5 minutos
   5. Tu ejecutable estará en: dist\CellSite\CellSite.exe

🐧 LINUX:
   1. Abre Terminal
   2. Navega a esta carpeta: cd /ruta/a/CellSite
   3. Ejecuta: bash build.sh
   4. Espera 3-5 minutos
   5. Tu ejecutable estará en: dist/CellSite/CellSite

═══════════════════════════════════════════════════════════════
  ⚙️ REQUISITOS
═══════════════════════════════════════════════════════════════

Para CREAR el ejecutable necesitas:
  • Python 3.8 o superior
  • Conexión a internet (solo para descargar dependencias)

Para USAR el ejecutable NO necesitas nada:
  • El .exe funciona solo
  • No necesita Python instalado
  • No necesita internet


═══════════════════════════════════════════════════════════════
  💡 CONSEJOS
═══════════════════════════════════════════════════════════════

✓ Crea el ejecutable en el sistema operativo donde lo usarás:
  - Windows → build.bat → CellSite.exe
  - Linux → build.sh → CellSite

✓ Una vez creado, copia toda la carpeta dist\CellSite\
  a USB o donde quieras usarlo

✓ Haz respaldos periódicos del archivo data\app.db
